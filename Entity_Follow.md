# Entity.Follow #
| **Syntax** | Entity.Follow(Entity) |
|:-----------|:----------------------|
| **Return** | nill |
| **Description** | Follow a position of Entity  |
| **Part of Library** | Entity |
| **Type** | Shared |
| **File** | core\sharedfiles\sh\_entity.lua |

## Examples ##
```
if Entity(2):IsNPC() then

   Entity(2):Follow(Entity(1))
end
```
